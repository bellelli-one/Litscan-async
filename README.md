# Разработка интернет-приложений (РИП) 2025

## Расчет вероятности авторства текста методом стилометрии

Разработанное приложение позволит читателям создавать и формировать заявки на расчёты вероятности авторства для выбранных ими книг, а библиографам завершать или отклонять данные заявки.
